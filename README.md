# Website of Shreejith Shanker, Assistant Professor, TCD. 
This repository contains the source for the website of Shreejith Shanker, Assistant Professor in Reconfigurable Computing, Trinity College Dublin. 

The code for this page is based on [al-folio](https://alshedivat.github.io/al-folio/) and is a simple, clean, and responsive [Jekyll](https://jekyllrb.com/) theme for academics/research pages.
